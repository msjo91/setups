# Install fonts
### Source Code Pro
+ Ubuntu

> Download

https://github.com/adobe-fonts/source-code-pro

> Unzip in `/usr/share/fonts/opentype`

```
sudo tar -xzf source-code-pro-variable-fonts.tar.gz
```

> Run command

```
sudo fc-cache -f -v
```

+ macOS

> Install

```
brew tap caskroom/fonts
brew install font-source-code-pro
```
